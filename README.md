# Text-Editor-Lanterna
Command Line Text Editor made in Java using Lanterna, a console text GUI library for Java.

## Key bindings:
* \<Esc\>     Exit <br>
* \<CTRL-s\> Save <br>
* \<Ctrl-z\> Undo <br>
* \<CTRL-r\> Mark <br>
* \<CTRL-c\> Copy <br>
* \<CTRL-x\> Cut <br>
* \<CTRL-\v> Paste
